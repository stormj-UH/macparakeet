# Itzhak Bentov Source Matrix

**Research value: high for devices/Medi-Tech, Flight 191, and Gateway provenance; moderate for biography and reception; low for suppression claims.** The hard sources are patents, Boston Scientific filings, the NTSB accident record, CIA/Army Gateway records, and publisher/library records. Several biographical and "Japanese scientists" details remain memoir- or secondary-source dependent.
